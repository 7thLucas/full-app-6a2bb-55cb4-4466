# Design Guidelines — InterviewIQ

## Colors
- Primary: Deep blue (#1E3A5F or similar)
- Accent: Electric teal (#00BCD4) for highlights and scores
- Background: Clean white / light gray
- Success: Green for high scores, amber for medium, red for low

## Typography
- Clean, professional sans-serif (Inter or similar)
- Large, readable score numbers
- Compact but scannable feedback cards

## Components
- Video player with timeline markers at key moments
- Score cards with radial/circular progress indicators
- Feedback panels per session segment
- Admin sidebar dashboard (use existing admin-sidebar pattern from CLAUDE.md)
- RBAC-aware views (admin vs candidate view)

## Elevation & Layout
- Card-based layout with subtle shadows
- Clean sidebar navigation
- Mobile-friendly responsive grid
